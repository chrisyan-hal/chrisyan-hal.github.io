G05.x 指令的語法與設定內容

chrisyan, 20170620

(release history)

G05.2 / G05.3 / G05.4 / G05.5 / G05.6 / G05.7 / G00拐角精度

 

G05.2 指令 (設定值會儘可能即刻生效, 不減速)

  F_ T_ - (單位 mm/min 與 ms) 指定一組程式速度與加減速時間關係.

        [註1: 原始設定為參數 120-124 號中, 所指定的程式速度與加減速時間關係.]

        [註2: T 的指定值如果小於 10ms, 或大於 2000ms, 會默認為其邊界值(即 10ms 或 2000ms.)]

        [註3: 假設原始設定有 2 組, 1000-100 與 2000-140 (mm/min-ms).  則下面的例子會在其間增加一組設定關係.

                G05.2 T120. F1500.;

            如果例子中是 F2500, 則會在 2 組之後增加第 3 組設定關係.]

        [註4: F 指定值小於等於 0, 或 T 沒有指定時, 不會增加此組程式速度與加減速時間關係.]

        [註5: 程式速度與加減速時間關係最多可以有 8 組設定, 當設定已滿時, 新增的設定會取代程式速度比它小
            且最接近的那一組.]

  T_ - (單位 ms) 指定一組程式速度與加減速時間關係的加減速時間, 而程式速度則使用最近一次指定的程式速度.

        [註1: 一個範例如下.

                G01 Z10. F3500;

                ...

                G05.2 T120;

            則會增加一組 3500-120 的設定.]
        [註2: 其餘注意事項同"F_T_"指令的內容.]
  J_ - (比例值) 實際使用的 Jerk 上限值與參數設定值的比值, 默認值為 1, 合法範圍 0.1-5 (超出則以邊界值設置).
        [註1: 原始 Jerk 上限值是設定於參數 151 號第 2 項(單位 0.1G/s).]
        [註2: 一個例子如下, 指定實際作用的 Jerk 最大值是參數所設的值的 1.2 倍.
                G05.2 J1.2; ]
  W_ - (無單位) 指定前饋控制的級別 (FeedForward Control Level).  默認值為 0, 即關閉前饋控制.
        [註1: W 的指定值如果小於 0, 則默認為 0; 如果大於 3, 則默認為 3.]
        [註2: 一個例子如下, 設定值會四捨五入到整數.
                G05.2 W2.8;
            這相當於指定"W3". ]
        [註3: 此項目前雖然可以指定, 但尚無作用.]
  Q_ - (0) 使用 Q0 取消 G05.2 的所有設定, 並回復第 1 個 G05.2 設定前的設定值.  其餘值目前沒有任何作用.
        [註: 範例如下.
                 G05.2 Q0; ]
[註1: 除了 G05.n 必須在其它設定值之前先指定外, 其餘設定可以不分次序, 範例中的空格也非必須, 以下均同.]
[註2: 以上設定可以分別逐次設置, 會分別生效, 直到"Q0"取消, 回復原設定狀態為止.  而每個 G05.2 都會規畫減速
    停止, 就像連續的 G01 遇到 G00 一樣.]

G05.3 指令 (設定值會儘可能即刻生效, 不減速)
  D_ - (單位 mm) 指定最大轉角誤差, 合法值範圍 1um-20um (超出則以邊界值設置), 但當設定值小於等於 0 時,
        默認值是 3um.
        [註: 原始值是參數 165 號第 1 項(單位 um).]
  E_ - (單位 mm) 指定特長直線間的最大轉角誤差, 設定值小於等於 0 時, 默認值同最大轉角誤差; 無最大值限制.
        [註1: 特長直線是指長度 5mm 或以上的直線.]
        [註2: 原始值是參數 125 號第 2 項(單位 um), 如果無設定或小於等於 0, 則使用上一項,(參數 165 號第 1 項)
            的設定值; 此參數值最大不超過 1234567.]
  C_ - (單位 mm) 指定 NURBS 擬合最大弦誤差, 合法值範圍 1um-10um (超出則以邊界值設置).
        [註: 原始值是參數 168 號第 1 項(單位 um).]
  H_ - (單位 mm) 指定在高速時 NURBS 擬合的最大弦誤差, 設定值小於等於 0 時, 默認值同最大弦誤差; 大於
        10um 時, 默認為 10um.  至於高速的設定, 請參見下一項.
  M_ - (單位 mm) 指定(外接)圓弧加點功能的最大弦誤差判斷值, 設定值小於等於 0 時, 默認值同最大弦誤差;
       大於 10um 時, 默認為 10um.
  F_ - (單位 mm/min) 指定高速的門檻值, 且設定值不小於默認值, 而默認值是 5000mm/min.
  S_ - (單位 mm) 指定 NURBS 擬合前, 移點的最大容許誤差, 合法值範圍 0.0001-0.01 (即 0.1-10um), 超過上限值
        時以上限值設置, 但低於下限值時(不含相等), 則以 0 處理, 即不移點.
        [註1: 原始值是參數 168 號第 2 項(單位 um).]
        [註2: 此項之處理必須在 NURBS 擬合移點功能開啟時才有效.]
  R_ - (單位 mm) 指定曲率半徑估計的最小值, 設定值最小不小於 10um, 默認是不限制.
        [註: 此項目前雖然可以指定, 但尚無作用.]
  P_ - (0, 1) 指定(外接)圓弧加點功能開啟(設為 1)或關閉(設為 0).
        [註: 原始值默認是開啟, 參數 150 號第 2 項的 Bit-9 設為 1 則關閉.]
  A_ - (0, 1) 指定連續小直線轉換為圓弧功能開啟(設為 1)或關閉(設為 0).
        [註: 原始值默認是關閉, 參數 150 號第 2 項的 Bit-3 設為 1 則開啟.]
  B_ - (0, 1) 指定(外接)圓弧加點功能開啟時, 使用可以加 3 點的方法(設為 1)或使用只加 1 點的方法(設為 0).
        [註: 原始值默認是使用只加 1 點的方法.]
  L_ - (0, 1) 指定連續小直線向以 Least Square 法(LSQ)逼近而得的圓弧, 進行移點的功能開啟(設為 1)或關閉
        (設為 0).
        [註1: 默認為關閉.]
        [註2: 此指令請不要與"擬合 NURBS 時對 Twist 區域則儘可能移點至以 Least Square 法求得的最佳圓弧上"
            的功能搞混了, 後者默認是開啟, 可以 150 號第 2 項參數的 Bit-8 設為 1 來關閉.]
        [註3: L1 與 L0 之間所夾的 G01 單節, 不考慮是否為小線段, 都進行 LSQ 圓弧移點.  但此功能一次處理的
            單節數限制, 目前為 50 個, 超過的會切割為另一次 LSQ 圓弧移點.  如果所夾的單節中有非 G01 者, 則
            處理會被切割為數次 LSQ 圓弧移點.]
        [註4: 每一個 LSQ 圓弧移點, 第 1 個單節的起點目前不會納入計算, 也不會被移點.]
        [註5: 除了以下一項的指令 K_ 或其默認值控制移點誤差外, 當其中某一點的計算值與原資料點偏差超過
            100um (目前為固定值)時, 則整段移點就放棄.]
        [註6: 移點的方法有 2 種, 默認是投影法, 就是要移到由計算所得圓心延伸線投影到圓弧面上的圓弧交點;
            另一種是半徑長度法, 只考慮由計算所得圓心延伸的長度, 調整到與計算所得圓弧半徑一致的方法.  而
            參數檔 150 號第 2 項參數的 Bit-11, 即為"Least Square (LSQ)後移點方式"的選項.  如果此 Bit 為默認值
            的 OFF, 則使用投影法, 反之如果為 ON, 則使用半徑長度法.]
  K_ - (單位 mm) 指定移點至以 Least Square 法求得的最佳圓弧上的最大容許誤差, 設定值小於等於 0 時, 默認
        值同 NURBS 擬合前移點的最大容許誤差, 大於 10um 時默認為 10um.
        [註: 此移點誤差設定值的原始值, 為參數 169 號第 2 項(單位 um).]
  I_ - (比例值) 小線段長度定義值與小線段長度基本定義值的比值, 合法範圍 0.1-10 (超出則以邊界值設置), 默認
       值為1.
        [註1: 每一單節均有其小線段長度基本定義值, 是以該單節的程式速度, 配合G01的加減速所計算出來的加
            減速距離.  如果以G01最大速度5m/min, 加減速時間120ms為例, 則其程式速度是2.5m/min的小線段長
            度基本定義值計算如下(以梯形加減速曲線為例, 即是加減速三角形的面積).
            程式速度^2 / (2 * 加減速) = (2.5m/min)^2 / ( 2 * (5m/min / 120ms) ) = 1.25mm ]
        [註2: 每一單節的小線段長度定義值, 是小線段長度基本定義值的數倍, 可由此項的參數設定方式改變其倍數
            (默認值是1倍). 每一單節的大線段長度定義值, 固定為小線段長度定義值的2倍.]
        [註3: 小線段單節定義為長度小於小線段長度定義值的單節(不含等於), 大線段單節定義為長度大於或等於大
           線段長度定義值的單節, 而單節長度介於1-2倍小線段長度定義值的(含1倍, 不含2倍), 暫稱為中線段.]
        [註4: 只有小線段單節才會通過篩選, 可被擬合為NURBS, 大線段與中線段單節則會被排除掉.]
  U_ - (單位 mm) 指定(外接)圓弧加點功能的最小弦誤差判斷值, 合法範圍 0.0005-0.01 (即 0.5-10 um, 超出則以邊
       界值設置), 默認值為 0.001 (即 1 um).
        [註: 此參數即Look-Ahead外接圓弧加點演算法中, 第 1 項第 6 子項所定義的 ECmin1.]
  V_ - (單位 mm) 指定(外接)圓弧加點功能的判斷條件中, 對應大線段的最小弦誤差值, 合法範圍 0.00005-0.005 (即
       0.05-5 um, 超出則以邊界值設置), 默認值為 0.0001 (即 0.1 um).
        [註: 此參數即Look-Ahead外接圓弧加點演算法中, 第 1 項第 6 子項所定義的 ECmin2.]
  W_ - (比例值) 指定直線合併功能的判斷條件中, 前後直線對其外接圓形成的弦誤差對加工精度(或轉角誤差)的比
       值, 默認值為 0.3, 合法範圍 0.01-1.0, 其中高於上限值以上限值設置, 但低於下限值(不含相等)時, 則以 0 處理, 即.
       不合併直線, 相當於直線合併功能關閉.
        [註1: 此參數設定值即Look-Ahead直線合併(Filter Aligned Blocks)演算法中, 第 1 項第 6-2 子項中所定義的 Ra.]
        [註2: 此項之處理必須在 直線合併功能開啟時才有效.]
  J_ - (比例值) 指定(外接)圓弧加點功能的判斷條件中, 檢查屬於平面線段的條件所使用的前後直線單節長度比值
       定義, 合法範圍 1-20 (超出則以邊界值設置), 默認值為 2.
        [註: 此參數即Look-Ahead外接圓弧加點"演算法中, 第 1 項第 6-1-2 子項所定義的 Rflat.]
  O_ - (2-6) 指定 NURBS 擬合方式的 NURBS 曲線階數(Degree).
       [註: 默認為 2 階(階數/Degree 2 即級數/Order 3), 指定值如超出合法範圍, 則使用邊界值.]
  Q_ - (0) 使用 Q0 取消 G05.3 的所有設定, 並回復第 1 個 G05.3 設定前的設定值.  其餘值目前沒有任何作用.
[註1: 以上設定可以分別逐次設置, 會分別生效, 直到"Q0"取消, 回復原設定狀態為止.  而每個 G05.3 的設定會儘
    可能即時生效, 而且除了設定造成的影響外, 不會特別規畫減速停止.]
[註2: 一個範例如下.
        G05.3 D0.005 E0.030 C0.005 H0.008 S0.003 F4500 R0.100 P1 A1 L1 K0.005; ]

 

G05.4 指令 (設定值會儘可能即刻生效, 不減速)
  A_ - (0, 1) 指定 NURBS 近似擬合(Approximation Fitting)方式啟動.
       [註: 默認為關閉.]
  B_ - (5-100) 指定 NURBS 近似擬合方式一次處理的最大單節數.
       [註: 默認最多 100 個單節.  只有在近似擬合啟動狀態下才接受命令, 指定值如超出合法範圍, 則使用邊界值.]
  D_ - (2, 3) 指定 NURBS 近似擬合方式的 NURBS 曲線階數.
       [註: 默認為 3 階.  只有在近似擬合啟動狀態下才接受命令, 指定值如超出合法範圍, 則使用默認值.]
  R_ - (0.5-1.0) 指定使用 NURBS 近似擬合方式時, 控制點與資料點數的比值.
       [註: 默認為 0.8.  只有在近似擬合啟動狀態下才接受命令, 指定值如超出合法範圍, 則使用邊界值.]
  O_ - (2-6) 指定 NURBS 近似擬合方式的 NURBS 曲線階數(Degree).
       [註: 默認為 3 階(階數/Degree 3 即級數/Order 4), 指定值如超出合法範圍, 則使用邊界值.]
  Q_ - (0) 使用 Q0 取消 G05.4 的所有設定, 並回復第 1 個 G05.4 設定前的設定值.  其餘值目前沒有任何作用.

 

G05.5 指令 (會在此指令前減速停止, 然後再繼續運動)
  L_ - (0-25) 指定SMF濾波強度 ( <=0 時關閉, >最大可接受的強度時指定此最大強度[目前昰25] ).
  S_ - (0, 1) 指定 SMF 濾波函數形狀 ( 0 為鐘形[Bell Shape],  1 為線性[Linear] ).
  Q_ - (0)  使用 Q0 取消 G05.5 的所有設定, 並回復第 1 個 G05.5 設定前的設定值.  其餘值目前沒有任何作用. ]
    [註1: 此指令會使速度規畫減速停止, 然後再啟動.]
    [註2: 此指令作用期間, 會忽略動態參數設定的命令(來自人機或 PLC), 直到 Q0 命令或 M30 或復位, 會解除 G05.5 的
    狀態, 並回復第 1 個 G05.5 設定前的設定值, 並接受動態參數設定的命令.]

 

G05.6 指令 (設定值會儘可能即刻生效, 不減速, 且目前只適用於每群組6軸版本)
  L_ - (0, 1) 指定對G43.4啟動中(工件座標上)的連續小直線, 以NURBS近似(NURBS Approximation)方法平滑化其端點

       刀軸向量(Tool Axis Orientation)的變化, 此功能的開啟(設為 1)或關閉(設為 0), 默認為關閉.
  I_ - (比例值) 用於平滑化各端點刀軸向量變化的功能, 指定(工件座標上)大線段長度定義值與小線段長度基本定義值
       的比值, 合法範圍0.1-100 (超出則以邊界值設置), 默認值為10.
        [註1: 每一單節均有其小線段長度基本定義值, 是以該單節的程式速度, 配合G01的加減速所計算出來的加
            減速距離.  如果以G01最大速度5m/min, 加減速時間120ms為例, 當其程式速度是2.5m/min時的小線段
            長度基本定義值計算如下(以梯形加減速曲線為例, 即是加減速三角形的面積).
            程式速度^2 / (2 * 加減速) = (2.5m/min)^2 / ( 2 * (5m/min / 120ms) ) = 1.25mm ]
        [註2: 每一單節的大線段長度定義值, 是小線段長度基本定義值的數倍, 可由此項的參數設定方式改變其倍數
            (默認值是10倍).]
  B_ - (0, 1, 2, 3) 指定以NURBS近似方法平滑化各端點刀軸向量的變化時, 各刀軸向量如何與NURBS曲線對應的方式.
        0, 使用以各刀軸向量近似擬合為NURBS時原始對應的節點(Knot)參數值(即u值). 此項為默認值.
        1, 目前暫與0相同.
        2, 使用各刀軸向量所對應(工件座標上)的端點, 其間的(XYZ)距離與總長度的比值, 而得到相對應的參數值(即u值).

        3, 只在工件座標上已擬合的NURBS中, 進行各端點刀軸向量的NURBS近似平滑化, 而參數值(即u值)則取自端點間
            (XYZ)距離與(有參與單節)總長度的比值.
        Others, 其它值設定的結果, 目前暫與0相同.
  K_ - (比例值) 用於限制平滑化各端點刀軸向量變化功能計算的結果, 即指定其NURBS參數值u的調整量, 不能超過距前
       或後一端點u值改變量的所設定比例值. 此設定的默認值為 0, 即不限制調整刀軸向量之u值的調整量; 最小設定值為

       0.001, 低於此值視為0, 最大設定值則沒有限制.
        [註1: 此設定只在設置B2或B3時才有作用.]

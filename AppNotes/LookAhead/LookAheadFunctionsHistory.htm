6) 定義圓弧加點後的弦誤差上限值為ECmax, 且令ECmax2 = 2 * ECmax. 另指定 ECmin1 = 1um 及 ECmin2 = 0.1um. 其中, ECmax默認為3um, 且可由G05.7的M指令變更; ECmin1與ECmin2分別可由G05.7的U指令與V指令變更其值.

Filter Aligned Blocks (接近直線刀路的合併)

1. 必須通過以下條件的檢查才進行第2項的處理.

1) 連續2個單節(B1及B2)均為直線命令(G01), 否則不處理.

2) 單節B1及B2均不為G31命令單節, 且不含第4軸命令, 否則不處理.

3) 單節B1及B2的長度分別為L1及L2, 且長單節(大線段)的長度定義為Lcr, 則必須L1, L2 < Lcr, 否則不處理.

4) 單節B1至B2與B2至下一單節的轉角分別為Q1與Q2, 則必須Q1 ≤ Qmax且Q2 ≤ Qmax, 否則不處理(其中Qmax由參數決定, 目前Qmax = 8⁰).

5) 如果單節B1已經合併了n個單節(含B1且n = 3), 則不處理.

6) 單節B1的起點P0至單節B2的終點P2的連線為L02, 且B1終點或B2起點為P1.

6-1) 計算求得P1對L02的垂線交點為Pa, 以及P1至Pa的距離L1a為B1與B2的外接圓的弦誤差.

6-2) 當加工精度(誤差)上限值為Eprec時, 則必須L1a ≤ Ra * Eprec (其中弦誤差對加工精度的比值上限為Ra = 0.6), 否則不處理.

6-3) 考慮B1與B2的合成長度為L02, 則必須L1a ≤ Rb * L02 (其中弦誤差對合成長度的比值上限為Rb = 0.005), 否則不處理.

6-4) 考慮B1與B2的合成長度為L02, 則必須L02 < Lcr, 否則不處理.

Filter Tiny Twisted Blocks (極小扭曲直線刀路的平滑化)

 

1. 必須通過以下條件的檢查才進行第2項的處理.

1) 連續3個單節(B1, B2及B3)均為直線命令(G01), 否則不處理.

2) 單節B1, B2及B3均不為G31命令單節, 且不含第4軸命令, 否則不處理.

3) 單節B1至B2與B2至B3的轉角分別為Q1與Q2, 則必須Qmin ≤ Q1, Q2 ≤ 90⁰, 否則不處理(其中Qmin = 5⁰).

4) 單節B2的長度L2, 且當加工精度(誤差)上限值為Eprec時, 必須L2 < Eprec, 否則不處理. [極小線段定義]

5) 單節B1至B3的轉角Q13, 則必須Q13 < Q1且Q13 < Q2, 否則不處理. [刀路扭曲(Twisted)定義]

 

2. 極小扭曲刀路的平滑化處理.

1) 取單節B2的中點為Pm.

2) 單節B1的終點改為Pm (即B2的前半段併入B1).

3) 單節B3的起點改為Pm (即B2的後半段併入B3).

4) 刪去單節B2.

1. 必須通過以下條件的檢查才進行第2項的處理.

1) 連續3個單節(B1, B2及B3)均為直線命令(G01), 否則不處理.

2) 單節B1, B2及B3均不為G31命令單節, 且不含第4軸命令, 否則不處理.

3) 單節B1, B2及B3的長度分別為L1, L2及L3, 且長單節(大線段)的長度定義為Lcr, 則必須L1, L2, L3 < Lcr, 否則不處理.

4) 單節B1至B2與B2至B3的轉角分別為Q1與Q2, 則必須Qmin ≤ Q1, Q2 ≤ Qmax, 否則不處理(其中Qmin = 0.5⁰, Qmax由參數決定, 目前Qmax = 8⁰).

5) 單節B1至B3的轉角Q13, 則必須Q13 < Q1且Q13 < Q2, 否則不處理. [刀路扭曲(Twisted)定義]

6) 取單節B2的中點為Pm, 把B2分為前半段與後半段.

6-1) 單節B1的起點P0至Pm的連線為L0m, 且B1的終點為P1, 則計算求得P1對L0m的垂線交點為Pa, 以及P1至Pa的距離L1a為B1與B2前半段的外接圓的弦誤差.

6-2) 單節B2中點Pm至B3的終點P3的連線為Lm3, 且B3的起點為P2, 則計算求得P2對Lm3的垂線交點為Pb, 以及P2至Pb的距離L2b為B2後半段與B3的外接圓的弦誤差.

6-3) 當加工精度(誤差)上限值為Eprec時, 則必須L1a < Eprec且L2b < Eprec, 否則不處理.

 

2. 一般扭曲刀路的平滑化處理.

1) 單節B1的終點及單節B2的起點改為Pa (即平滑化處理B1與B2的前半段).

2) 單節B2的終點及單節B3的起點改為Pb (即平滑化處理B2的後半段與B3).

 

 

Filter Aligned Blocks (接近直線刀路的合併)

 

1. 必須通過以下條件的檢查才進行第2項的處理.

1) 連續2個單節(B1及B2)均為直線命令(G01), 否則不處理.

2) 單節B1及B2均不為G31命令單節, 且不含第4軸命令, 否則不處理.

3) 單節B1及B2的長度分別為L1及L2, 且長單節(大線段)的長度定義為Lcr, 則必須L1, L2 < Lcr, 否則不處理.

4) 單節B1至B2與B2至下一單節的轉角分別為Q1與Q2, 則必須Q1 ≤ Qmax且Q2 ≤ Qmax, 否則不處理(其中Qmax由參數決定, 目前Qmax = 8⁰).

5) 如果單節B1已經合併了n個單節(含B1且n = 6), 則不處理.

6) 單節B1的起點P0至單節B2的終點P2的連線為L02, 且B1終點或B2起點為P1.

6-1) 計算求得P1對L02的垂線交點為Pa, 以及P1至Pa的距離L1a為B1與B2的外接圓的弦誤差.

6-2) 當加工精度(誤差)上限值為Eprec時, 則必須L1a ≤ Ra * Eprec (其中弦誤差對加工精度的比值上限為Ra = 0.6), 否則不處理.

6-3) 考慮B1與B2的合成長度為L02, 則必須L1a ≤ Rb * L02 (其中弦誤差對合成長度的比值上限為Rb = 0.005), 否則不處理.

 

2. 接近直線刀路的合併處理.

1) 單節B1的終點P1改為單節B2的終點P2.

2) 刪去單節B2.

 

 

Add Circumcircle Points (外接圓弧加點)

 

1. 必須通過以下條件的檢查才進行第2項的處理.

1) 連續3個單節(B1, B2及B3)均為直線命令(G01), 否則不處理.

2) 單節B1, B2及B3均不為G31命令單節, 且不含第4軸命令, 否則不處理.

3) 單節B2或B3是NURBS的一部分, 或B2的任一端點已經是加點的結果, 則不處理.

4) 單節B1, B2及B3的長度分別為L1, L2及L3, 且長單節(大線段)的長度定義為Lcr, 則必須L2 < Lcr, 否則不處理.

5) 承上一項, L1 < Lcr或L3 < Lcr (即L1與L3不同時大於等於Lcr), 否則不處理.

6) 當加工精度(誤差)上限值為Eprec時, 定義Emax = 2 * Eprec.

6-1) 當L1 < Lcr且L3 < Lcr (即B1與B3同時不為大線段)時.

6-1-1) 必須L1, L2, L3 > Emax, 否則不處理.

6-1-2) 當L2/L1 > 10且L2/L3 > 10時, 則不處理. [平面(Flat)區段定義]

6-1-3) 單節B1與B2的外接圓在B2上所造成的弦誤差為C12Ec2, 則必須C12Ec2 ≥ 1, 否則不處理. 當C12Ec2 > Eprec時, 令C12Ec2 = Eprec. 而由C12Ec2的值, 可求得弦誤差點C12Pec2.

6-1-4) 單節B2與B3的外接圓在B2上所造成的弦誤差為C23Ec2, 則必須C23Ec2 ≥ 1, 否則不處理. 當C23Ec2 > Eprec時, 令C23Ec2 = Eprec. 而由C23Ec2的值, 可求得弦誤差點C23Pec2.

6-1-5) 由6-1-3及6-1-4項求得的弦誤差點C12Pec2及C23Pec2, 取2者平均而得平均弦誤差點Pec2, 及平均弦誤差Ec2.

6-2) 當L1 ≥ Lcr且L3 < Lcr (即B1為大線段而B3否)時.

6-2-1) 必須L2, L3 > Emax, 否則不處理.

6-2-2) 單節B1與B2的外接圓在B2上所造成的弦誤差為C12Ec2, 則必須C12Ec2 ≥ 0.1, 否則不處理.

6-2-3) 考慮與單節B1相切的單節B2的外接圓, 在B2上所造成的弦誤差為C2Ec2, 可求得弦誤差點C2Pec2.

6-2-4) 單節B2與B3的外接圓在B2上所造成的弦誤差為C23Ec2, 則必須C23Ec2 ≥ 1, 否則不處理. 當C23Ec2 > Eprec時, 令C23Ec2 = Eprec, 並求得修正的弦誤差點C23Pec2.

6-2-5) 由6-2-3及6-2-4項求得的弦誤差點C2Pec2及C23Pec2, 取2者平均而得平均弦誤差點Pec2, 及平均弦誤差Ec2.

6-3) 當L1 < Lcr且L3 ≥ Lcr (即B3為大線段而B1否)時.

6-3-1) 必須L1, L2 > Emax, 否則不處理.

6-3-2) 單節B1與B2的外接圓在B2上所造成的弦誤差為C12Ec2, 則必須C12Ec2 ≥ 1, 否則不處理. 當C12Ec2 > Eprec時, 令C12Ec2 = Eprec, 並求得修正的弦誤差點C12Pec2.

6-3-3) 單節B2與B3的外接圓在B2上所造成的弦誤差為C23Ec2, 則必須C23Ec2 ≥ 0.1, 否則不處理.

6-3-4) 考慮與單節B3相切的單節B2的外接圓, 在B2上所造成的弦誤差為C2Ec2, 可求得弦誤差點C2Pec2.

6-3-5) 由6-3-2及6-3-4項求得的弦誤差點C12Pec2及C2Pec2, 取2者平均而得平均弦誤差點Pec2, 及平均弦誤差Ec2.

6-4) 由6-1-5或6-2-5或6-3-5項可求得平均弦誤差點Pec2及平均弦誤差值Ec2. 而當Ec2 > Emax時, 不處理. 否則當Ec2 > Eprec時, 令Ec2 = Eprec, 並求得修正的弦誤差點Pec2.

 

2. 外接圓弧加點處理.

1) 增加一個新單節B2n, 其起點為前述檢查條件6-4項求得的弦誤差點Pec2, 終點為單節B2的終點(P2).

2) 單節B2的終點改為弦誤差點Pec2, 成為新的B2單節nB2 (起點維持為P1).

3) 如果新B2單節nB2 (或新單節B2n)的長度是L2new, 當L2new < Lnew時, 結束處理程序(其中Lnew = 0.2mm). 此時原單節B2, 經由加點Pec2成為2個單節nB2及B2n.

4) 當nB2 (或B2n)的長度是L2new, 如果L2new ≥ Lnew時, 執行以下子項目, 繼續加2點.

4-1) 由新B2單節nB2與新單節B2n的外接圓, 取新B2單節nB2的外接圓部分, 得到其圓弧中點(即弦誤差點)Pec2A. 增加一個新單節nB2n, 其起點為Pec2A, 終點為Pec2. 修改新B2單節nB2的終點為Pec2A, 成為單節n2B2(起點維持為P1).

4-2) 取新單節B2n的外接圓部分, 得到其圓弧中點(即弦誤差點)Pec2B. 增加一個新單節B2n2, 其起點為Pec2B, 終點為P2. 修改新單節B2n的終點為Pec2B, 成為單節nB2n (起點維持為Pec2).

4-3) 此時原單節B2, 經由加3點Pec2A, Pec2及Pec2B, 成為4個單節n2B2, nB2n, nB2n及B2n2.

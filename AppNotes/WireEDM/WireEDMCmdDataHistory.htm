Heterogeneous Interpolation*

[Command] (valid only when heterogeneous interpolation on)

G01/G02/G03 X_ Y_ I_ J_ R_ G01/G02/G03 U_ V_ K_ L_ R_ Z_ F_

  G01/G02/G03(1st) - Interpolation command on program plane

  X_ Y_ - positions of X and Y axes on program plane

  I_ J_ - relative positions of center from starting point on program plane

  R_(1st) - radius of cornering with next block on program plane

  G01/G02/G03(2nd) - Interpolation command on auxiliary plane (only when heterogeneous interpolation on)

  U_ V_ - relative positions of X and Y axes on auxiliary plane from X_ Y_

  K_ L_ - relative positions of center from starting point on auxiliary plane

  R_(2nd) - radius of cornering with next block on auxiliary plane

  Z_ - position of Z axis

  F_ - federate

[Data Format]

  gcode0 = 1/2/3 for G01/G02/G03(1st),  subcode = 1/2/3 for G01/G02/G03(2nd)

  end0.x = dX_,  end0.y = dY_,  end0.z = dZ_,  end0.a = dU_,  end0.b = dV_  (as least input increment)

  cen0.x = I_,  cen0.y = J_,  cen0.z = K_,  radius = R_(1st),  frate = F_  (as least input increment)

  op2 = L_,  op3 = R_(2nd)  (as least input increment)

  abc.x = X_,  abc.y = Y_,  abc.z = Z_,  abc.a = U_,  abc.b = V_  (as least input increment)

  ax_flag = fX_,  ay_flag = fY_,  az_flag = fZ_,  aa_flag = fU_,  ab_flag = fV_

  ai_flag = fI_,  aj_flag = fJ_,  ak_flag = fK_,  ar_flag = fR_(1st),  af_flag = fF_

  aop2_flag = fL_,  aop3_flag = fR_(2nd),  reserved2 = 161

Specification of Commands and Their Outputs

 

  Nomenclature:

       X_ - command value for command (address) X

       dX_ - incremental command value for command (address) X

       fX_ - flag of command appearance and type for command (address) X

  Unit:

    least input increment for Metric – 0.0001 mm

    least input increment for Inch – 0.0001 inch

 

Rapid Positioning

[Command]

G00 X_ Y_ U_ V_ Z_

  X_ Y_ - positions of X and Y axes on program plane

  U_ V_ - positions of X and Y axes on auxiliary plane

  Z_ - position of Z axis

[Data Format]

  gcode0 = 0,  subcode = 0

  end0.x = dX_,  end0.y = dY_,  end0.z = dZ_,  end0.a = dU_,  end0.b = dV_  (as least input increment)

  abc.x = X_,  abc.y = Y_,  abc.z = Z_,  abc.a = U_,  abc.b = V_  (as least input increment)

  ax_flag = fX_,  ay_flag = fY_,  az_flag = fZ_,  aa_flag = fU_,  ab_flag = fV_

  U_ V_ - positions of X and Y axes on auxiliary plane

Circular/Helical CW/CCW Interpolation

[Command]

G02/G03 X_ Y_ I_ J_ U_ V_ K_ L_ Z_ F_ R_,R_

  X_ Y_ - positions of X and Y axes on program plane

  I_ J_ - relative positions of center from starting point on program plane

  U_ V_ - positions of X and Y axes on auxiliary plane

  K_ L_ - relative positions of center from starting point on auxiliary plane

  Z_ - position of Z axis

  F_ - federate

  R_ - radius of cornering with next block on program plane

  ,R_ - radius of cornering with next block on auxiliary plane

[Data Format]

  gcode0 = 2/3,  subcode=0

  end0.x = dX_,  end0.y = dY_,  end0.z = dZ_,  end0.a = dU_,  end0.b = dV_  (as least input increment)

  cen0.x = I_,  cen0.y = J_,  cen0.z = K_,  radius = R_,  frate = F_  (as least input increment)

  op2 = L_,  op3 = ,R_  (as least input increment)

  abc.x = X_,  abc.y = Y_,  abc.z = Z_,  abc.a = U_,  abc.b = V_  (as least input increment)

  ax_flag = fX_,  ay_flag = fY_,  az_flag = fZ_,  aa_flag = fU_,  ab_flag = fV_

  ai_flag = fI_,  aj_flag = fJ_,  ak_flag = fK_,  ar_flag = fR_,  af_flag = fF_

  aop2_flag = fL_,  aop3_flag = f,R_,  reserved2 = 60/61
